# Датасет: поток клиентов в отделения банка

## Описание

Синтетический датасет, моделирующий работу трёх банковских отделений за 4 недели (июнь 2025). Содержит информацию о визитах клиентов, типах операций, сотрудниках и параметрах отделений.

## Файлы

### client_arrivals.csv (~6 400 записей)
Каждая строка — один визит клиента.

| Поле | Описание |
|---|---|
| arrival_datetime | Дата и время прихода (YYYY-MM-DD HH:MM) |
| weekday | День недели (Пн, Вт, ..., Сб) |
| operation_id | Тип операции (OP01–OP06) |
| service_time_min | Фактическое время обслуживания (мин) |
| branch_id | Идентификатор отделения (BR01–BR03) |

### operations.csv (6 записей)
Справочник типов операций.

| Поле | Описание |
|---|---|
| operation_id | Идентификатор |
| name | Название операции |
| avg_service_min | Среднее время обслуживания (мин) |
| std_service_min | Стандартное отклонение времени обслуживания |
| min_service_min | Минимальное время обслуживания |
| frequency_weight | Относительная частота операции |
| required_skill | Требуемый навык сотрудника (basic / credit / mortgage) |

### employees.csv (22 записи)
Справочник сотрудников.

| Поле | Описание |
|---|---|
| employee_id | Идентификатор |
| branch_id | Привязка к отделению |
| name | Имя |
| grade | Грейд (junior / middle / senior) |
| skills | Навыки через запятую |
| max_hours_week | Максимум часов в неделю |
| max_hours_day | Максимум часов в день (включая обед) |
| lunch_min | Длительность обеденного перерыва (мин) |
| hourly_cost_rub | Стоимость часа работы (руб) |

### branches.csv (3 записи)
Параметры отделений.

| Поле | Описание |
|---|---|
| branch_id | Идентификатор |
| name | Название |
| type | Тип (flagship / standard / small) |
| n_windows | Число окон обслуживания |
| hours_weekday | Часы работы пн-пт |
| hours_saturday | Часы работы сб |
| hours_sunday | Часы работы вс |

### hourly_load.csv
Агрегированная нагрузка: число клиентов и суммарное время обслуживания по каждому часу каждого дня.

## Подсказка

Поток клиентов хорошо описывается пуассоновским процессом с переменной интенсивностью. Интенсивность зависит от часа дня и дня недели.
